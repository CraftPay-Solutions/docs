## Pobierz plugin

Pobierz najnowszą wersję pluginu CraftPay dla swojego serwera Minecraft.

<Card title="Pobierz CraftPay Plugin" icon="download" href="https://github.com/craftpay-solutions/minecraft-plugin/releases/latest">
  Kliknij tutaj, aby pobrać najnowszą wersję pluginu (.jar)
</Card>

<Info>
  Plugin wspiera serwery **Spigot**, **Paper** oraz **Bukkit** w wersjach **1.8 - 1.21\+**.
</Info>

## Wymagania

Przed instalacją upewnij się, że posiadasz:

- Serwer Minecraft (Spigot / Paper / Bukkit)
- Java 17 lub nowsza
- Konto na [panel.craftpay.pl](https://panel.craftpay.pl) oraz klucz API

## Instalacja

<Steps>
  <Step title="Pobierz plugin">
    Pobierz plik `.jar` z linku powyżej.
  </Step>
  <Step title="Wgraj na serwer">
    Umieść plik `CraftPay.jar` w folderze `plugins/` swojego serwera.
  </Step>
  <Step title="Uruchom serwer">
    Zrestartuj serwer, aby plugin wygenerował domyślną konfigurację w `plugins/CraftPay/config.yml`.
  </Step>
  <Step title="Skonfiguruj klucz API">
    Otwórz `config.yml` i wklej swój klucz API ze strony [panel.craftpay.pl](https://panel.craftpay.pl).

    ```yaml
    api-key: "TWÓJ_KLUCZ_API"
    ```
  </Step>
  <Step title="Przeładuj plugin">
    Wykonaj komendę `/craftpay reload`, aby zastosować zmiany.
  </Step>
</Steps>
