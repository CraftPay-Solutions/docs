</Info>

## Wymagania

Przed instalacją upewnij się, że posiadasz:

- Serwer Minecraft (Spigot / Paper / Bukkit)
- Java 17 lub nowsza
- Konto na [app.craftpay.pl](https://panel.craftpay.pl) oraz klucz API

## Instalacja

<Steps>
  <Step title="Pobierz plugin">
    Pobierz plik `.jar` z linku powyżej.
  </Step>
  <Step title="Wgraj na serwer">
    Umieść plik `craftpay-1.0.0.jar` w folderze `plugins/` swojego serwera.
  </Step>
  <Step title="Wgraj na serwer">
    Umieść plik `craftpay-1.0.0.jar` w folderze `plugins/` swojego serwera.
  </Step>
  <Step title="Wgraj na serwer">
    Umieść plik `craftpay-1.0.0.jar` w folderze `plugins/` swojego serwera.
  </Step>
  <Step title="Uruchom serwer">
    Zrestartuj serwer, aby plugin wygenerował domyślną konfigurację w `plugins/CraftPay/config.yml`.
  </Step>
  <Step title="Skonfiguruj klucz API">
    Otwórz `config.yml` i wklej swój klucz API ze strony [app.craftpay.pl](https://panel.craftpay.pl).

    ```yaml
    api_secret: "TWOJ_SEKRET_KEY"
    ```
  </Step>
  <Step title="Przeładuj plugin">
    Wykonaj komendę `/craftpay reload`, aby zastosować zmiany.
  </Step>
</Steps>
